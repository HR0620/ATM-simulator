# 音声ファイルの配置について

現在、以下の **MP3ファイル** がコードに対応しています。
`assets/sounds/` フォルダに配置してください。

1. **open-window.mp3**
   - 顔認証成功時に再生

2. **irassyaimase.mp3**
   - ATM画面表示時に再生

3. **pushenter.mp3**
   - ボタン選択時（ジェスチャー認識時）に再生

4. **come-again.mp3**
   - アプリ終了時（ESCキー）に再生
   - ※お持ちのファイル名が `come-agein.mp3` の場合は、`come-again.mp3` にリネームしてください（スペル修正）。

※ 以下のファイルはお持ちのようですが、現在の画面フローではまだ使用していません。
- check-money.mp3
- pay-money.mp3
